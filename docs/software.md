  - [MalwareBytes](https://www.malwarebytes.com/) - antivirus

  - [Proton](https://protonvpn.com/) - VPN  

::: info One can use package manager(s) like [winget](https://learn.microsoft.com/en-us/windows/package-manager/winget/#install-winget), [scoop](https://scoop.sh/), [chocolatey](https://chocolatey.org/install) for updating apps like mpv, 7zip, vlc, hugo, etc

package managers automate installation, updates, and dependency handling, saving time and effort
+
they allow bulk installs and easy version control, which manual .exe or .msi downloads can’t.

if you have to choose one, I would go with scoop, make sure its maintained by genuine source and is updated often.
:::
