# Games

:::warning Required Components

Install these three before downloading games (legitimate or pirated) to avoid crashes from missing software on your computer:

- [DirectX](https://www.microsoft.com/download/details.aspx?id=35)
- [VisualCppRedist AIO](https://github.com/abbodi1406/vcredist/releases/latest)
- [XNA Framework](https://www.microsoft.com/download/details.aspx?id=20914)
:::

## Repacks

::: details What are repacks?
Repacks are compressed games for low-bandwidth users, but installing them takes time due to file decompression.
:::

## Direct Download Sites

Direct downloads are normal downloads from a server, being safer and not requiring a VPN. You may need a VPN to access blocked file hosts (like Rapidgator in some EU countries). I would recommend using [AB Download Manager](https://abdownloadmanager.com/) to speed up downloads.
